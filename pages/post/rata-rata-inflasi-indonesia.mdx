# Berapa rata-rata inflasi di Indonesia

Inflasi rata-rata tahun 2002 sampai Juni 2022: 5.2


Saya selalu penasaran berapa sih sebenarnya rata-rata inflasi di Indonesia.

Coba saya cari di google tetapi tidak memuaskan.

<Image px={20} src="/google-search-rata-rata-inflasi.png" mb={5} />

Tapi saya lihat di search result saya melihat ada raw data dari [BI](https://www.bi.go.id/id/statistik/indikator/data-inflasi.aspx)

Setelah saya liat terdapat data dari 2003 sampai 2022, karena datanya tidak begitu mudah untuk diproses (tahun dan percent nya berbentuk string), perlu dilakukan preprocessing data.

Selengkapnya bisa dilihat di [github](https://github.com/faisalburhanudin/rata-rata-inflasi-indonesia) saya.

Dari hasil analysis bisa dilihat rata rata inflasi setiap tahun adalah 5.2
